Resumen de GitHub

GitHub es una plataforma para el alojamiento de código fuente y control de versiones mediante Git. Permite a los desarrolladores colaborar en proyectos de manera efectiva, llevar un historial de cambios, y contribuir a proyectos de código abierto.

  • git clone [URL]: Clona un repositorio remoto en tu máquina local.
  • git commit -m "mensaje": Registra cambios locales con un mensaje descriptivo.
  • git push origin [rama]: Envía tus cambios a la rama especificada en el repositorio remoto.
  • git pull: Actualiza tu copia local con los cambios del repositorio remoto.

  • Usa ramas para trabajar en nuevas características sin afectar la rama principal.
  • Confirma los cambios frecuentemente para un historial claro y organizado.
  • Colabora mediante Pull Requests para revisar cambios antes de fusionarlos.
